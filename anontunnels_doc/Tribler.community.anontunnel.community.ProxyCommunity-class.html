on_data(self, circuit_id, candidate, message)

source code 

Handles incoming DATA message

Determines whether the data comes from the outside world (origin set) or whether the data came from the origin (destination set)

If the data comes from the outside world the on_incoming_from_tunnel method is called on the observers and the circuit is marked as active

When the data comes from the origin we need to EXIT to the outside world. This is left to the observers as well, by calling the on_exiting_from_tunnel method.

Parameters:
  • int, circuit_id - the circuit's id we received the DATA message on
  • Candidate|None, candidate - the messenger of the packet
  • DataMessage, message - the message's content
Returns:
whether the message could be handled correctly
