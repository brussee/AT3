_try_request(self)

source code 

Try to consume a REQUEST message and respond whether we will accept the request.

Will setup a TCP relay or an UDP socket to accommodate TCP RELAY and UDP ASSOCIATE requests. After a TCP relay is set up the handler will deactivate itself and change the Connection to a TcpRelayConnection. Further data will be passed on to that handler.

:return: None
